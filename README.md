# STM32 Rainwater Harvesting System

Smart rainwater harvesting and monitoring system based on STM32.

## Features

### Implemented
- Water level measurement
- Distance measurement using HC-SR04
- Tank fill percentage calculation
- Tank volume calculation
- UART monitoring

### Planned
- DS18B20 temperature measurement
- LCD display
- ESP8266 WiFi monitoring
- Data logging
- Mobile notifications

## Hardware

### Current
- STM32F103C8T6
- HC-SR04 ultrasonic sensor
- USB-UART adapter

### Planned
- DS18B20 temperature sensor
- ESP8266 WiFi module
- LCD display

## Software

- STM32CubeIDE
- STM32 HAL
- KiCad
- GitHub
