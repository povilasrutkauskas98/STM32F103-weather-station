# STM32 Rainwater Harvesting System

Smart rainwater harvesting and monitoring system based on STM32.

## Features

- Water level measurement
- Tank volume calculation
- Rainfall monitoring
- Temperature measurement
- LCD display
- WiFi monitoring
- Mobile notifications
- Data logging

## Hardware

- STM32F103C8T6
- HC-SR04 ultrasonic sensor
- DS18B20 temperature sensor
- ESP8266 WiFi module
- LCD display

## Software

- STM32CubeIDE
- STM32 HAL
- KiCad
- GitHub
